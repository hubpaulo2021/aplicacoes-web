41 - Calcule no JS e mostre em um script

Para calcular dois números, siga as instruções abaixo: Clique no botão abaixo para calcular os números. Ao clicar no botão "Clique aqui", você será solicitado a inserir o primeiro número em um prompt. Após confirmar, um segundo prompt será exibido para que você digite o segundo número. O resultado da soma dos dois números será mostrado em seguida. Dicas: Utilize o editor de código VSCode para criar o arquivo HTML. Lembre-se que é um Script externo. Estilize o CSS externo. Entrega: Salve o arquivo como exercicio41.html e compartilhe ou envie para o classroom.
