42 - Multiplique no JS e mostre em um script

Para multiplicar os três números, siga as instruções abaixo: Clique no botão abaixo para multiplicar os números. Ao clicar no botão "Clique aqui", você será solicitado a inserir o primeiro número em um prompt. Após confirmar, um segundo prompt será exibido para que você digite o segundo número e em seguida vai solicitar o terceiro número. O resultado da multiplicação dos três números será mostrado em seguida. Dicas: Utilize o editor de código VSCode para criar o arquivo HTML. Lembre-se que é um Script externo. Estilize o CSS externo.
