Exercício 55 - Calculadora de Operações Matemáticas Simples

Desenvolva uma página em HTML com dois campos de entrada para que o usuário insira dois números. Adicione quatro botões separados para realizar as operações de soma, subtração, multiplicação e divisão. Quando o usuário clicar em um dos botões, o resultado da operação correspondente deve ser exibido diretamente na página.
